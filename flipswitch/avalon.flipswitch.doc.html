flipswitch

滑动式开关 flipswitch组件，将checkbox表单元素转化成富UI的开关，不支持ms-duplex，请在onChange回调里面处理类似ms-duplex逻辑
使用说明
名字默认值说明
配置参数
toggle true 组件是否显示，可以通过设置为false来隐藏组件
onText "" 选中状态提示文字
offText ""//"×" 未选中状态提示文字
size "normal" 滑动条类型，默认normal，可设置为large,small,mini，以及其他任意组件不自带的名词，可以用来注入自定义class，生成ui-flipswitch-{{size}}添加给flipswitch模板容器
theme "normal" 主题，normal,success,warning,danger
draggable false 是否支持拖动切换状态
disabled false 禁用
checked false 默认选中状态
animated true 是否开启切换动画效果
hdir true 开启、关闭选项排列顺序默认为true，即on-off,false为off-on
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
_animateArrMaker配置接口_animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
onChange回调函数onChange(newValue, vmodel) 选中状态发生变化时触发，参数为当前的选中状态及vmodel对象
getTemplate配置接口getTemplate(tmpl, opts) 用于修改模板的接口，默认不做修改
toggleStatus接口调用toggleStatus 交替改变选中状态
disable接口调用disable 禁用组件
enable接口调用enable 启用组件
_draw接口调用_draw() 动态更换皮肤后，可以调用这个方法更新提取switch样式

例子：

  1. 不同大小size，默认设置，选中状态
  2. 不同大小size，支持可拖动切换，选中状态
  3. 不同大小size，OFF-ON顺序排列，选中状态
  4. 不同大小size，disabled状态，选中状态
  5. 不同大小size，自定义ON-OFF文本，选中状态
  6. 不同大小size，通过接口修改状态，选中状态
  7. flipswitch + tooltip
