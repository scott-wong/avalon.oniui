menu

菜单组件 现扫描dom元素或者设置传参生成级联菜单的组件，注意扫描dom情形下，会销毁原有的dom，且会忽略所有的ol，ul，li元素上原有的绑定
使用说明
名字默认值说明
配置参数
activefalse 将第几个项目设置为选中，级联情形下，会将设置应用给每一级menu，默认是false，一个都不选中，建议不要通过修改这个值来修改menu的选中状态，而是通过setActiveList接口来做
data undefined menu的数据项，如果没有配置这个项目，则默认扫描元素中的li，以及li中的ul或者ol来创建级联菜单，数据结构形式 
[
{
title: "html",
 data: [...],
 active: false,
 disabled: false
}
]
，子元素如果包含有效的data属性表示拥有子菜单
event "mouseenter" 选中事件，默认mouseenter
dir "h" 方向，取值v,h，默认h是水平方向， v是竖直方向
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
menuResetter配置接口menuResetter(vmodel) 选中某个menu项之后调用的这个restter，默认是把menu重置为不选中
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
onSelect回调函数onSelect(vmodel, realSelect, _hasSubMenu) this指向选中的menu li元素，realSelect是选中menu项目的数组 
[
[data, active],
[data2,active2]
]
，对应每一级的数据，及每一级的active值，_hasSubMenu表示this元素有无包含子menu
activate接口调用activate(index)展开菜单索引为index的项目，index置为false,undefined则不会展开任一项目
getActiveList接口调用getActiveList() 获取所有选中的menu list
setActiveList接口调用setActiveList(activeListArray) 设置级联menu的选项，可以一个数组，也可以使一个数字，或者"2,3,4"这样的字符串
