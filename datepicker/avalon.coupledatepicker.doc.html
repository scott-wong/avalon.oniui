coupledatepicker

日期范围选择的双日历 coupledatepicker其实是普通日历的升级版，可以通过设置起始日期与结束日期的最小间隔天数和最大间隔天数将截止日期限制在一定的选择范围中
使用说明
名字 默认值 说明
配置参数
fromCalendarLabel "选择起始日期" 设置起始日期选择日历框的说明文字
toCalendarLabel "选择结束日期" 设置结束日期选择日历框的说明文字
disabled false 设置是否禁用组件
startDay 1 设置每一周的第一天是哪天，0代表Sunday，1代表Monday，依次类推, 默认从周一开始
separator "-" {{type: String}} 日期格式的分隔符，可以是"/"或者你希望的符号，但如果是除了"-"和"/"之外的字符则需要和parseDate和formatDate配合使用，一边组件能正常运作
接口方法与固有属性
rules "" {{type: String | Object}} 设置双日历框的工作规则 
                            {
                                rules: 'null, 0D, 8D',
                                fromMinDate: '2014-05-02',
                                fromMaxDate: '2014-06-28',
                                toMinDate: '2014-06-01',
                                toMaxDate: '2014-07-12'
                            }
可以是绑定组件时定义的配置对象中的一个rules对象，也可以是一个字符串，指向一个上述对象。
其中对象中的rules属性定义结束初始日期异常时默认显示的日期、初始日期和结束日子之间最小相隔天数、最大相隔天数。
格式是[+-]\d[DMY]，分别代表几天、几个月或者几年，也可以附加+或者-号，+号表示正数几天，-号表示负数几天 

                            fromMinDate代表起始日期可以设置的最小日期
                            fromMaxDate代表起始日期可以设置的最大日期
                            toMinDate代表结束日期可以设置的最小日期
                            toMaxDate代表结束日期可以设置的最大日期
parseDate(str) 将符合日期格式要求的字符串解析为date对象并返回，不符合格式的字符串返回null,用户可以根据自己需要自行配置解析过程 
    parseDate : function(str){
        var separator = this.separator;
        var reg = "^(\\d{4})" + separator+ "(\\d{1,2})"+ separator+"(\\d{1,2})$";
        reg = new RegExp(reg);
        var x = str.match(reg);
        return x ? new Date(x[1],x[2] * 1 -1 , x[3]) : null;
    },
formatDate(date) 将日期对象转换为符合要求的日期字符串，默认实现如下： 
    formatDate : function(date){
        var separator = this.separator,
            year = date.getFullYear(), 
            month = date.getMonth(), 
            day = date.getDate();
        return year + separator + formatNum( month + 1 , 2 ) + separator + formatNum( day , 2 );
    },
