scrollbar

滚动条 scrollbar组件，自定义滚动条样式，绑定ms-widget="scrollbar"的元素内必须包含一个class="ui-scrollbar-scroller"的视窗元素
使用说明
名字默认值说明
配置参数
disabled false 组件是否被禁用，默认为否
toggle true 组件是否显示，可以通过设置为false来隐藏组件
position "right" scrollbar出现的位置,right右侧，bottom下侧，可能同时出现多个方向滚动条
limitRateV 1.5 竖直方向，拖动头最小高度和拖动头宽度比率
limitRateH 1.5 水平方向，拖动头最小宽度和高度的比率
scrollTop 0 竖直方向滚动初始值，负数会被当成0，设置一个极大值等价于将拖动头置于bottom
scrollLeft 0 水平方向滚动初始值，负数会被当成0处理，极大值等价于拖动头置于right
show "always" never一直不可见，scrolling滚动和hover时候可见，always一直可见
showBarHeader true 是否显示滚动条两端的上下箭头
draggerHTML "" 滚动条拖动头里，注入的html碎片
size "normal" srollbar size,normal为10px，small为8px，large为14px
breakOutCallback配置接口breakOutCallback(["h", "up"], vmodel) 滚动到极限位置的回调，用来实现无线下拉等效果 breakOutCallback(["h", "up"], vmodel) 第一个参数是一个数组，分别是滚动条方向【h水平，v竖直】和超出极限的方向【up是向上或者向左，down是向右或者向下】，第三个参数是一个对象，包含滚动条的元素，宽高等信息
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
viewHeightGetter配置接口viewHeightGetter(viewElement) 配置计算视窗高度计函数，默认返回innerHeight
viewWidthGetter配置接口viewWidthGetter(viewElement) 配置计算视窗宽度计函数，默认返回innerWidth
getTemplate配置接口getTemplate(tpl, opts) 定制修改模板接口
onScroll回调函数onScroll(newValue, oldValue, diretion, vmodel) 滚动回调,scrollLeft or scrollTop变化的时候触发，参数为newValue, oldValue, diretion, vmodel diretion = h 水平方向，= v 竖直方向
getBars接口调用getBars()返回所有的滚动条元素，avalon元素对象
getScroller接口调用getScroller()返回scroller avalon对象
update接口调用update()更新滚动条状态，windowresize，内容高度变化等情况下调用，不能带参数
scrollTo接口调用scrollTo(x,y) 滚动至 x,y

例子：

  1. scrollbar-document.body带上下小箭头
  2. scrollbar-元素带上下小箭头双滚动条
  3. scrollbar-带上下小箭头嵌套
  4. scrollbar-元素无上下小箭头双滚动条
  5. scrollbar-滚动条hover或者滚动时候可见，初始不可见
  6. scrollbar-滚动条不可见
