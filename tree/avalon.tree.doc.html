tree

tree组件

tree组件，借鉴ztree实现的avalon版本树组件，尽量接近ztree的数据结构

使用说明
名字默认值说明
配置参数
view{Object} 视觉效果相关的配置
view.showLine true view.showLine是否显示连接线
view.dblClickExpand true view.dblClickExpand是否双击变化展开状态
view.selectedMulti true view.selectedMulti true / false 分别表示 支持 / 不支持 同时选中多个节点
view.showIcon true view.showIcon zTree 是否显示节点的图标
view.showTitle true view.showTitle 分别表示 显示 / 隐藏 提示信息
callback{Object} 回调相关的配置
view.nameShower配置接口view.nameShower(leaf)节点显示内容过滤器，默认是显示leaf.name
callback.onExpand配置接口callback.onExpand(data) 节点展开回调
callback.onCollapse配置接口callback.onCollapse(data) 节点收起回调
callback.onSelect配置接口callback.onSelect(data) 节点被选中回调
callback.onClick配置接口callback.onClick(data) 节点被点击回调
callback.onDblClick配置接口callback.onDblClick(data) 节点被双击回调
onInit回调函数onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate配置接口getTemplate(tpl, opts, tplName) 定制修改模板接口
expand接口调用expand(leaf, all) 展开leaf节点的子节点，all表示是否迭代所有子孙节点
collapse接口调用collapse(leaf, all) 折叠leaf节点的子节点，all表示是否迭代所有子孙节点
moveNode接口调用moveNode(targetLeaf, leaf, moveType, isSilent)
targetNode：移动节点要移动到的目标节点；
leaf：被移动的节点；
moveType：指定移动到目标节点的相对位置，"inner"：成为子节点，"prev"：成为同级前一个节点，"next"：成为同级后一个节点；
isSilent：设定移动节点后是否自动展开父节点。isSilent = true 时，不展开父节点，其他值或缺省状态都自动展开。
freeSelect接口调用freeSelect(event, leaf)取消leaf节点上所有处于选中状态的节点
avalon.ui.tree.AddExtention接口调用avalon.ui.tree.AddExtention(fixNames, addingDefaults, addingMethodFunc, watchEvents)扩展tree
